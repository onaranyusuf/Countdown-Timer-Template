## Project Setup
```
npm install
```
### Creating project
```
vue create "file name"
```
### Change directory
```
cd "file name"
```

### Add vuetify framework
```
vue add vuetify
```
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
